## Water Level Indicator 

## Why I made it 
I made this project for learning because I want to implement this type of indicator in my home water tank at home when the tank gets filled and no one checks it so the water overflows and gets wasted so we have to check it manually and to fix that I wanted to make a similar type of project using an ESP32 microcontroller that is going to be my next project this project was for learning how water level indicators work and next time I want to make one on my own without the help of a tutorial.

## What is it 
Water Level Indicator is a simple hardwrae project. It uses a water level sensor connected to a arduino uno... when water reaches a certain threshhold like in my code i used 600 as thresh-hold value so whenever water level will increase from 600 the buzzer will beep and LED's will turn on.

https://youtu.be/CspfUhOipF0

## How to use it 
**Step 1**: Connect the Water level sensor to the arduino Uno
.VCC → 5V
.GND → GND
.Signal → A0
**Step 2**: Connect the buzzer to Arduino
.Positive terminal → Digital Pin 9/11/7/8 (can be any of them)
.Negative terminal → GND
**Step 3**: Connect the LED 
.Positive terminal → Digital Pin 8/11/7/6/5 (can be any of them) using a 220Ω resistor
.Negative terminal → GND
**Step 4**: Upload the program to the Arduino Uno
**Step 5**: Place the water level sensor where it can measure the water level of the water
